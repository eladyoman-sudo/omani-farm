# 🌾 تطبيق الزراعة العماني
**Omani Agricultural Application**

## 📖 نظرة عامة

تطبيق زراعي متطور لسلطنة عمان يساعد المزارعين والتجار على:
- ✅ **معرفة المحاصيل** المناسبة لكل محافظة
- ✅ **جدولة الزراعة والحصاد** طوال السنة  
- ✅ **تحديد مصادر المياه** (أفلاج وآبار)
- ✅ **سوق زراعي مباشر** للبيع والشراء
- ✅ **مستشار زراعي ذكي** (AI)

---

## 🚀 المميزات

### 🔐 مصادقة قوية
- تسجيل دخول بالبريد الإلكتروني أو الهاتف
- كلمات مرور محفوظة بـ **bcrypt**
- sessions آمنة مع localStorage

### 🗺️ خريطة تفاعلية
- 11 محافظة عمانية
- مصادر المياه التقليدية (الأفلاج)

### 📚 دليل المحاصيل
- 20+ محصول موسمي
- جداول الزراعة والحصاد

### 🛒 السوق الزراعي
- قائمة المنتجات للبيع والشراء
- التقييمات والمراجعات

---

## 🛠️ التقنيات

\\\
Frontend: Next.js 16 + React 19 + TypeScript + Tailwind CSS
Backend: Node.js + API Routes + MongoDB
Authentication: Email/Phone + bcryptjs
Deployment: Vercel + GitHub
\\\

---

## ⚡ البدء السريع

### 1. تثبيت
\\\ash
npm install
\\\

### 2. إعداد البيانات
\\\
MONGODB_URI=mongodb+srv://...
NEXTAUTH_SECRET=your-key
\\\

### 3. التشغيل
\\\ash
npm run dev
\\\

اذهب إلى: **http://localhost:3000** 🎉
